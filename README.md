## 설정 관련 주요 내용
각 링크에 설명이 포함되어 있으니 읽어보시기를 추천드립니다.
- 각종 모듈화와, 설정 편의를 위해 [Klippain]()을 사용
- 이름은 위험하지만, 전혀 위험하지 않은 [DangerKlipper]()를 사용
- 다수의 MCU를 편하게 업데이트하기 위한 [UKAM]() 사용
- 그리고, 설정 파일을 이곳에 백업하기 위해 [BackupConfigToGithub]()사용
